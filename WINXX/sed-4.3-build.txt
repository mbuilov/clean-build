How to build native sed.exe for Windows

From cygwin or other unix shell:

1) get archive:      wget http://ftp.gnu.org/gnu/sed/sed-4.3.tar.xz
2) unpack archive:   tar xf sed-4.3.tar.xz
3) go to sources:    cd sed-4.3
4) fix sed-4.3-windows.patch, if needed:

to build with Visual Studio/WDK version other than
"C:\Program Files (x86)\Microsoft Visual Studio 14.0"
"C:\Program Files (x86)\Windows Kits\10\Include\10.0.14393.0"

it is needed to fix sed-4.3-windows.patch - to change paths to Visual Studio/WDK,
for example, if Visual Studio is installed in "C:\opt\Studio 2008", use the following commands:

sed -i '/Windows Kits/s@C:.*ucrt@C:\\opt\\Studio 2008\\VC\\include@' sed-4.3-windows.patch
sed -i '/Visual Studio/s@C:.*include@C:\\opt\\Studio 2008\\VC\\include@' sed-4.3-windows.patch

5) patch sources:    patch -Np1 -i ../sed-4.3-windows.patch
6) run dos prompt:   cmd.exe /c "start cmd.exe"
7) setup compiler:   "c:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\vcvarsall.bat" x86_amd64
8) do compile:       make.bat
9) build result:     sed\sed.exe
